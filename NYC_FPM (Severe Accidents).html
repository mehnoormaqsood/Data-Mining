Motor Vehicle Accidents in New York City¶

In a developed city like New York, the competitive and fast-paced environment offers its own consequences. One of them is increased vehicular traffic and consequently, more accidents and crashes. While some accidents are inevitable, road safety and policies backed up by analysis can lead to apt suggestions about how to curb accidents or at least avoid them. This will be the core motivation of this project. In this report, we analyze the prevalence of vehicular accidents in NYC – their causes, effects and trends throughout the city. There will also be some recommendations about key insights for variables to consider that could shape future analysis.

# Using library implementation of Apriori Algorithm
from apyori import apriori

def apriori_alg(transactions, minimum_support):
    ret_dict = {}
    frequent_itemsets = apriori(transactions, min_support = minimum_support)
    frequent_itemsets = list(frequent_itemsets)
    freq_itemsets = [(list(frequent_itemsets[i][0]),frequent_itemsets[i][1]) for i in range(len(frequent_itemsets))]
    
    for (s, support) in freq_itemsets:
        ret_dict[tuple(s)] = int(support*len(transactions))
        
    #return freq_itemsets
    return ret_dict

def apriori_algo(transactions, minimum_support,minimum_length):
    # Make frequent patterns
    freq_patterns  = apriori_alg(transactions, minimum_support)

    # Choose frequent pattern of required length (or greater)
    temp = freq_patterns.copy()
    for i in temp:
        if len(i) < minimum_length:
            freq_patterns.pop(i)
    
    return freq_patterns

def freqitemset_list(freqitemset_dictionary):
    freq_pattern = []
    for tup in freqitemset_dictionary.keys():
        freq_pattern.append(list(tup))
